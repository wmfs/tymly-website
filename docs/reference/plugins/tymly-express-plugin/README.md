# tymly-express-plugin

::: tip Description
Exposes the Tymly framework via an Express.js web app.
:::

## Links

* [Github repository](https://github.com/wmfs/tymly-express-plugin#readme)
* [NPM package](https://www.npmjs.com/package/@wmfs/tymly-express-plugin)





## Blueprint directories
| Directory | Description |
| --------- | ----------- |
| `/images` | A place to put images that can be served-up in Forms and similar |




## Services
| Name | Description |
| ---- | ----------- |
| [jwtAuth](services/jwt-auth.html) | Provides JWT-based authentication capabilities |
| [server](services/server.html) | Provides a CORS-enabled Express.js server. Includes serving of static assets as supplied via Blueprints |
| [stateboxApi](services/statebox-api.html) | Exposes the core Tymly abilities (start, get, update and cancel) over a JWT-secured REST API |
